# Instructions

## Compile and run
### `python` code
`python burgers.py`

### `c++` code

Compile use `g++ -o burgers burgers.cpp`.
Run with `./burgers`. (Authorize by `chmod +x burgers`)

### `CUDA` code

Compile using 
`nvcc -arch=sm_50 burgers.cu -o test`.
Run with `./test`.
